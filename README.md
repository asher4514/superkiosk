## Quickstart
1: `git clone https://github.com/ashermyers/superkiosk`
<br >
2: `yarn` (Install Dependencies).
<br >
3: Edit Config in main.js

## Building
1: `yarn add electron-builder`
<br >
2: `yarn app:dist`
<br >
3: `cd dist`
<br >
4: `snap install superkiosk_2.0.0_amd64.snap --dangerous`

## Starting
`$ superkiosk`
